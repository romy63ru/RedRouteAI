# 🛰️ RedRouteAI

**Welcome to Mars. We have rovers. And AI. And logistics. Also bugs.**

RedRouteAI is a playground project for testing out new and shiny .NET and Microsoft tech in a *comically serious* setting:  
**Martian logistics with autonomous rovers, real-time tracking, and GPT-powered planning.**

---

## 🚀 What is this?

A full-stack Clean Architecture .NET app that lets you:
- Create and track **cargo orders** between **Martian bases**
- Watch **rovers** move (pretend) in real-time on a map
- Ask an AI assistant questions like “Where the heck is my cargo?”
- Experiment with **MediatR, SignalR, EF Core**, and **OpenAI** — without blowing up prod

---

## 💡 Why does this exist?

Because you love:
- trying **Clean Architecture** again (this time for real)
- testing **CQRS, AI integration, and DDD** with minimal consequences
- building things you can actually show your team without them asking “Is this production?”

This is **not** a startup, a product, or a moonshot. It’s Mars. And it’s yours.

---

## 🛠️ Tech Stack (a.k.a. excuses to avoid JavaScript)

- `.NET 8`, `ASP.NET Core`
- `Blazor Server` (or something frontendy if you're bold)
- `EF Core` (InMemory for now)
- `MediatR` + CQRS
- `SignalR` (for rover telemetry ⚡)
- `OpenAI` (for answering human questions about fake cargo)
- `Docker`, `GitHub Actions`, maybe `Azure` if you’re feeling spicy
