# RedRouteAI

**RedRouteAI** — демо-проект логистики на Марсе с ИИ-поддержкой, чат-ботом и визуализацией перемещения роверов между базами.

## Возможности
- Создание заказов между базами
- Отслеживание маршрутов в реальном времени
- Чат с ИИ по статусу грузов
- Оптимизация маршрутов через OpenAI

## Технологии
- ASP.NET Core Web API
- Blazor Server UI
- SignalR + OpenAI
- Docker + GitHub Actions
